# UPI Transactions Data Analysis Dashboard (Power BI)

## Project Overview
This project focuses on analyzing UPI transaction data using Power BI. The dashboard helps visualize transaction trends and user insights through interactive reports and data modeling techniques.

## Objectives
- Analyze UPI transaction data
- Build an interactive dashboard for better data insights
- Apply data modeling and visualization techniques

## Tools & Technologies
Power BI  
Data Modeling  
Excel Dataset  

## Key Features
- Imported and loaded UPI transaction dataset into Power BI
- Performed data profiling to understand dataset structure
- Used table view for data exploration
- Created an Age Group column for better demographic analysis
- Added multiple report pages for better visualization
- Created Matrix Visual for data analysis
- Applied Conditional Formatting to highlight key metrics
- Implemented Bookmarks for interactive dashboard navigation

## Visualizations Used
- Matrix Visual
- Charts
- Slicers
- Conditional formatting
